# 수련용사 키우기 (가제)

스팀(PC) 패키지 게임 프로젝트. 이 문서는 초기 기획 요약본으로, 개발이 진행되며 계속 갱신될 예정입니다.

## 1. 핵심 콘셉트 (Core Identity)

- **한 줄 정의**: PC 모니터 한 구석(백그라운드)에서 실행되는 '사무직/개발자 맞춤형 타자 감응형 힐링 RPG'.
- **핵심 재미**: 유저의 실제 업무(타이핑)가 게임 속 용사의 성장(수련)으로 다이렉트 연동되는 싱크로율과 시각적 카타르시스.
- **지향점**: 복잡한 BM(과금 구조) 스트레스 없는 $2.99~$4.99 선의 알찬 스팀 패키지 게임.

## 2. 주요 기능 및 시스템 (Key Features)

- **윈도우 투명화 및 테두리 없는 창 (Borderless/Transparent)**: 모니터 구석에 바탕화면을 투과하여 상시 상주하는 초경량 그래픽 뷰.
- **글로벌 키보드 후킹**: 게임 창이 내려가 있어도(비활성화) 유저의 실제 키보드 타수를 완벽히 감지하는 로직.
- **듀얼 모드 시스템**:
  - 수련 모드 (방치/업무 중): 허수아비를 치며 안전하게 경험치를 쌓고 기본 체급을 올리는 모드.
  - 탐험 모드 (도전/휴식 중): 정해진 난이도의 던전(웨이브 방식)에 들어가 몬스터를 처치하고 핵심 재화(골드, 장비 등)를 파밍하는 모드.
- **타이핑 콤보 & 연출 스케일링**:
  - 연속 타이핑 시 콤보 카운트 작동 (보상 밸런싱 공수 배제).
  - 콤보 수에 따라 검기, 화면 흔들림 등 이펙트 크기를 코드로 확장시켜 시각적 도파민 극대화.
  - 일정 콤보 달성 시 위트 있는 직장인 컨셉의 '타이틀(칭호)' 보상 지급.
- **라이프 사이클 연동 (퇴근 및 스케줄러 시스템)**:
  - 유저가 설정한 퇴근 시간이 되면 용사도 오늘 수련을 마치고 여관/주점으로 이동해 휴식하는 감성 연출.
  - 낮(자동 루틴)과 밤(유저 직접 조작 버프 제공 등)의 플레이 경험 분리.
- **마을 재건 및 건물 해금 (Town Building)**:
  - 모은 재화로 마을 UI 내 건물(대장간, 여관, 분수대, 교회 등)을 해금.
  - 각 건물 건설 시 무기 업그레이드, 퇴근 시스템, 랜덤 이벤트 스폰 확률 증가 등 상위 콘텐츠가 락(Lock) 해제되는 장기 목표 제공.

## 3. 게임 진행 플로우 (Core Loop)

```
[출근/업무 시작] ➔ [수련 모드 켜기] ➔ 열심히 타이핑 (업무/코딩)
       ↓
[콤보 폭발!] ➔ 콤보 카운트 상승 및 이펙트 화려하게 펑펑 (스트레스 해소)
       ↓
[경험치/골드 축적] ➔ 축적된 체급으로 [탐험 모드 (던전)] 진입 ➔ 장비/강화석 파밍
       ↓
[마을 정비 (UI 팝업)] ➔ 건물 해금 및 대장간에서 장비 구매/강화 (외형 간지 상승)
       ↓
[퇴근 시간 알람] ➔ 용사도 여관으로 퇴근, 하루 정산 보고서 팝업 및 휴식 전환
```

## 4. 개발 전략 (Asset-Driven)

- 기획서를 완벽히 뽑고 개발하는 대신, 확보 가능한 3D 에셋(캐릭터 모션, 무기 모델링, 허수아비 등)을 먼저 배치한 뒤 그 비주얼에 맞춰 기획 수치와 연출을 역산.
- 클로드 코드를 활용해 초반에 [투명창 세팅 + 글로벌 후킹 + 콤보 타이머]라는 핵심 기술 스파이크(프로토타입)를 빠르게 검증하여 개발 리스크 최소화.
