# CopperAxeBarbarian — Character Brief

> 상태: C형 2등신 + B형 장비 가독성의 PerfectPixel 재시험 후보
> 작업명이며 최종 캐릭터 이름은 Master Design 승인 전 변경할 수 있다.

## 기본 정보

- ID / 작업명: `CopperAxeBarbarian`
- Role: Player
- 게임 안의 기능: 양손의 한손 도끼로 빠른 연속 타격을 표현하는 근접 공격 캐릭터
- 성격 키워드: 호전적, 강인함, 야성적
- 한 문장 콘셉트: 구리빛 피부와 전투 문신을 지닌 근육질의 쌍도끼 바바리안 전사
- 승인 시점: 전면이 비교적 넓게 보이는 3/4뷰. 얼굴과 시선은 화면 오른쪽을 향함
- 기본 화면 진행 방향: 화면 오른쪽
- 좌우 기준: 수식어 없는 오른쪽/왼쪽은 캐릭터 자신의 해부학적 기준

## 체격과 실루엣

- 게임 내 상대 크기: 아직 미확정. PerfectPixel 출력 후 캐릭터별 표시 배율로 검증
- 전체 비율: 전체 높이가 머리 약 2개인 귀여운 마스코트형 2등신
- 체형: 짧고 둥근 몸통 안에서 넓은 어깨, 굵은 팔과 다리가 읽히는 근육질 체형
- 실루엣 핵심:
  1. 큰 머리와 사각형에 가까운 짧고 넓은 상체
  2. 높이와 각도가 다른 좌우 한손 도끼 두 자루
  3. 맨살 상체 위로 읽히는 굵은 팔·가슴 문신
- 기본 자세: 양발을 짧게 벌리고 무게중심을 낮춘 귀여운 전투 대기 자세
- 전방 디딤발: 화면 진행 방향인 화면 오른쪽에 더 가까운 발

## 얼굴과 머리

- 성별: 남성
- 피부: 따뜻한 구리빛 갈색
- 머리: 검은색 전사식 땋은 머리. 옆머리는 두피를 따라 정돈해 땋고, 뒤쪽은 실루엣이 읽히는 굵은 브레이드로 묶음
- 머리 길이: 굵은 뒤쪽 브레이드는 목 아래~등 상단 범위에 머물며 도끼와 팔 실루엣을 가리지 않음
- 얼굴: 굵은 눈썹, 강한 턱선, 짧고 거친 검은 수염 또는 턱수염
- 표정: 전투를 앞둔 단단한 인상, 과장된 분노 표정은 기본 Idle에서 제외

## 의상과 장비

- 상체: 의복 없음. 가슴, 복부와 팔 근육이 읽혀야 함
- 어깨: 낡은 철제 소형 어깨보호대 1개. 몸통 실루엣을 과도하게 넓히지 않음
- 어깨보호대 고정: 넓은 갈색 가죽끈이 보호대에서 반대쪽 허리 방향으로 맨가슴을 사선으로 가로지르고,
  겨드랑이 아래 보조끈과 연결되는 실제 착용 가능한 구조. 피부에 붙어 떠 있는 형태 금지
- 팔: 갈색 가죽 아대 또는 손목 보호대
- 하체: 짙은 갈색 가죽/모피 허리띠와 전투용 하의, 짧은 부츠
- 주 무기: 양손에 동일 계열의 한손 전투도끼 각 1자루
- 도끼 규격: 손잡이 길이 캐릭터 신장의 약 38%, 도끼날 폭 신장의 약 18%
- 도끼 배치: 앞손과 뒷손 도끼의 높이와 각도를 달리해 두 자루가 구분되도록 함
- 금지 장비: 투구, 상체 갑옷, 방패, 양손 대형 도끼, 망토

## 문신

- 색: 짙은 청흑색
- 위치: 노출된 팔, 어깨와 가슴 한쪽을 잇는 굵은 기하학적/부족풍 패턴
- 형태: 작은 글자나 복잡한 선이 아니라 게임 크기에서도 읽히는 2~4개의 굵은 덩어리
- 문신은 흉터, 발광 룬, 페이스 페인팅으로 변형하지 않음

## 픽셀 스타일

- 방향: 승인된 전면 3/4 시점을 유지하고 얼굴과 시선은 화면 오른쪽을 향함
- 스타일: KeyBuddy 데스크탑 컴패니언에 맞는 귀여운 저밀도 SD 픽셀 아트. 체격과 쌍도끼 실루엣은 묵직하게 표현
- 컨셉 입력 해상도: 논리 해상도로 강제 축소하지 않고 가장 선명한 생성 원본을 사용
- 생산 픽셀 밀도: 현재 미확정. PerfectPixel 512 출력과 실제 게임 표시 결과로 결정
- 외곽선 방향: 어두운 갈색/청흑색의 굵고 선명한 계단형 외곽선
- 팔레트: 작은 화면에서 피부·머리·가죽·금속이 명확히 분리되는 제한 팔레트 지향
- 광원: 좌상단
- 명암: 각 재질당 명확한 2~3단계, 부드러운 그라데이션 금지
- 안티앨리어싱, 반투명 가장자리와 고해상도 페인팅 질감 금지

## 고정 팔레트 방향

- 피부: 구리빛 중간색 + 밝은 황갈색 하이라이트 + 짙은 적갈색 그림자
- 머리/수염: 검정에 가까운 청흑색
- 문신: 피부 그림자와 구분되는 차가운 청흑색
- 가죽: 짙은 적갈색과 황갈색
- 금속: 저채도 회색, 차가운 밝은 회색 하이라이트
- 하의/모피: 어두운 갈색과 회갈색

정확한 팔레트 값은 선택된 Master Design에서 추출해 고정한다.

## 절대 바뀌면 안 되는 요소

- 구리빛 피부의 근육질 남성
- 검은 머리와 짧은 검은 수염
- 옆머리와 뒤쪽이 명확히 땋인 검은 브레이드 헤어
- 맨살 상체
- 굵고 어두운 부족풍 문신
- 양손에 동일 계열의 한손 도끼 한 자루씩
- 소형 어깨보호대와 가죽 아대
- 가슴 사선 가죽끈으로 고정된 어깨보호대 구조
- 낮고 안정된 전투 자세

## 금지 요소

- 뿔 달린 투구와 판타지 왕관
- 상체를 덮는 갑옷이나 셔츠
- 고정끈 없이 맨살에 붙어 있는 어깨보호대
- 한 자루뿐인 도끼 또는 양손으로 드는 대형 도끼
- 지나치게 사실적이거나 2.5등신 이상으로 길어진 인체 비율
- 과도한 장신구, 해골 장식과 발광 마법 효과
- 지면, 접지 그림자, 배경, UI, 글자
- 무기나 신체가 캔버스 밖으로 잘리는 구도

## 1차 후보 평가 순서

1. 두 도끼가 명확히 서로 구분되는가
2. 구리빛 피부, 검은 머리와 문신이 작은 화면에서도 분리되는가
3. 맨살 상체의 근육이 SD 픽셀 형태로 읽히는가
4. C형 2등신의 귀여움을 유지하면서 바바리안의 무거운 실루엣이 읽히는가
5. 전방 디딤발 접촉점과 안전 여백이 명확한가

## PerfectPixel Character & Style 입력

상세한 위 Brief는 내부 검수용이다. PerfectPixel에는 아래 값만 복사한다.

```text
Character name: CopperAxeBarbarian
Character description: Cute two-head-tall copper-skinned barbarian with braided black hair, a beard, a bare tattooed torso, a diagonal leather harness and a small iron guard on his right shoulder. He holds one matching one-handed axe in each hand; preserve his compact proportions and front-biased three-quarter view.
Art style: Pixel Art
Frame cell size: 512 x 512 px
Facing direction for animations: Not set
```

업로드 기준 이미지는 아래 파일을 사용한다.

```text
ProjectDocs/ArtPipeline/Characters/CopperAxeBarbarian/References/CopperAxeBarbarian-master-input-v3-c-hybrid-512.png
```

이 파일의 패딩은 캐릭터 크기를 통제하기 위한 값이 아니다. PerfectPixel이 콘텐츠를 다시 확대하는 동작은
이미 확인됐으므로 512 출력에서 품질을 우선 확보하고, 상대 체격과 애니메이션별 축척은 출력 이후 검증한다.
